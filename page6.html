RE-QUANTIZATION

RIGHT THE FIRST TIME

I got sick of looking at the previous page and decided that I wanted to do quantization properly.

I looked up some quantization tutorials but discovered that apparently what I did first was actually pretty much exactly how quantization is usually done, so it just looked basically the same but a little worse, actually.

So then I did it how I wanted to do it and created a set palette that I matched my image to, which is what you see here.

ANALYSIS

I won't say that this looks better necessarily, but I think it at least looks more intentional than normal quantization. There's something about a set color palette that makes it interesting to look at.

I could have spent more time findind a suitable color palette (I think I could have chosen another gray tone and changed the dark green tone) but this works just fine as a proof of concept/proof I can do it.

I also got better at defining variables since I had to define a vec4 array to define all my colors, which nice practice if nothing else.
